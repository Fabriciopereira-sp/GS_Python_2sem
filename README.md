Sobre o projeto

O SkillShift AI é um sistema em Python que realiza operações CRUD completas (Create, Read, Update e Delete) para gerenciar profissionais e trilhas de aprendizado, utilizando um banco de dados Oracle da FIAP.

O programa funciona totalmente pelo terminal e foi desenvolvido com foco educacional, atendendo aos requisitos da Global Solution de Computational Thinking Using Python.

Tecnologias utilizadas

Python 3

Banco de Dados Oracle

Biblioteca oracledb

Manipulação de arquivos JSON

Dependências necessárias

Antes de rodar o projeto, você precisa ter:

Python 3 instalado

Biblioteca oracledb instalada

Oracle Instant Client (caso o sistema operacional exija)
